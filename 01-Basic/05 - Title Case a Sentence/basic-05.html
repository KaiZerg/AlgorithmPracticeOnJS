Title Case a Sentence

Return the provided string with the first letter of each word capitalized. Make sure the rest of the word is in lower case. For the purpose of this exercise, you should also capitalize connecting words like "the" and "of".

Type in commands below into browser console, to see the result:

  • titleCase("I'm a little tea pot") should return a string.
  • titleCase("I'm a little tea pot") should return "I'm A Little Tea Pot".
  • titleCase("sHoRt AnD sToUt") should return "Short And Stout".
  • titleCase("HERE IS MY HANDLE HERE IS MY SPOUT") should return "Here Is My Handle Here Is My Spout".

My solution:

function titleCase(str) {
    var myArray = str.toLowerCase().split(' '); //convert the sentence to lower case and split it into an array
    for(var i=0;i        myArray[i] = myArray[i].charAt(0).toUpperCase() + myArray[i].substring(1);
    }
    myArray = myArray.join(' ');
    return myArray;
}

What did i learn?

  • While iterating through the array, for each string I converted the first character to upper case using the Array.charAt() method and toUpperCase method, then using the substring method take out the string excludung the first character and concatenate the first character to this string, this will produce a string with the first character in upper case
